# ClickFlow — черновик Privacy Policy

ClickFlow работает локально на устройстве пользователя.

## Какие данные используются

Приложение может использовать:

- Accessibility Service для выполнения нажатий по экрану;
- Overlay permission для отображения плавающих меток;
- MediaProjection для временного захвата экрана при поиске картинки или текста;
- выбранные пользователем изображения/иконки как локальные шаблоны.

## Что сохраняется на устройстве

Локально могут сохраняться:

- позиции меток;
- настройки таймера и повторов;
- выбранные шаблоны изображений;
- настройки поиска текста.

## Что не отправляется

ClickFlow не отправляет данные на сервер.

- Скриншоты не загружаются в облако.
- Шаблоны изображений не отправляются разработчику.
- OCR и поиск картинки выполняются локально на устройстве.

## Разрешения

### Accessibility Service

Нужен только для выполнения пользовательских автоматических нажатий.

### Overlay

Нужен только для отображения плавающих меток поверх других приложений.

### Screen capture / MediaProjection

Нужен только для поиска картинки или текста на экране.

## Контакты

Добавить контакт разработчика перед публикацией.
