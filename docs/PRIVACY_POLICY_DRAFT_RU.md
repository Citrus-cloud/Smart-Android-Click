# ClickFlow — черновик Privacy Policy

ClickFlow — Android-приложение для локальной автоматизации нажатий. Приложение работает на устройстве пользователя и не использует собственный сервер.

## Кратко

- ClickFlow не продаёт и не передаёт пользовательские данные третьим лицам.
- Скриншоты, шаблоны изображений, OCR-результаты, сценарии и настройки обрабатываются локально на устройстве.
- ClickFlow не загружает экран, изображения или сценарии на сервер разработчика.
- Android Backup для данных приложения отключён, чтобы локальные шаблоны и сценарии не копировались в системный облачный бэкап.

## Какие данные использует приложение

ClickFlow может использовать следующие данные только для работы выбранных пользователем функций:

- координаты нажатий и позиции overlay-меток;
- настройки интервалов, повторов и бесконечного режима;
- пользовательские сценарии автоматизации;
- выбранные пользователем изображения/иконки как локальные шаблоны для поиска на экране;
- текстовый запрос для OCR-поиска;
- временный снимок экрана для поиска картинки или текста.

## Что сохраняется на устройстве

Локально могут сохраняться:

- настройки ClickFlow;
- позиции меток;
- шаблоны изображений для режима «клик по фото»;
- настройки режима «клик по тексту»;
- созданные пользователем сценарии;
- флаг разблокировки тестового премиум-режима.

Эти данные нужны, чтобы настройки и сценарии сохранялись между запусками приложения.

## Что не сохраняется и не отправляется

ClickFlow не отправляет данные на сервер разработчика.

- Скриншоты не загружаются в облако.
- Шаблоны изображений не отправляются разработчику.
- OCR и поиск картинки выполняются локально на устройстве.
- ClickFlow не собирает пароли, платёжные данные, личные сообщения или другую чувствительную информацию.
- ClickFlow не использует рекламу и стороннюю аналитику в текущей версии.

## Разрешения

### Accessibility Service

Accessibility Service нужен для выполнения пользовательских нажатий, запуска созданных пользователем сценариев и локального получения снимка экрана на Android 11+ там, где это доступно системно.

Служба специальных возможностей используется только после ручного включения пользователем в настройках Android и только для функций, которые пользователь сам запускает внутри ClickFlow.

ClickFlow не использует Accessibility для чтения паролей, платёжных данных, личных сообщений или скрытого сбора информации.

### Overlay / Отображение поверх других приложений

Разрешение Overlay нужно для отображения плавающих меток, панели управления и кнопки остановки поверх других приложений.

### Screen capture / MediaProjection

Захват экрана нужен для поиска картинки или текста на экране. Android показывает системное подтверждение перед началом захвата. Снимки экрана используются временно для локального анализа и не отправляются на сервер.

### Notifications / Уведомления

На Android 13+ приложение может запросить разрешение на уведомления, чтобы показывать статус работающих режимов и повысить стабильность фоновой работы.

## Хранение и удаление данных

Пользователь может удалить данные приложения стандартными средствами Android:

1. Настройки Android → Приложения → ClickFlow.
2. Хранилище → Очистить данные / Очистить кэш.

Также можно удалить приложение — локальные данные ClickFlow будут удалены вместе с ним.

## Дети

ClickFlow не предназначен специально для детей и не собирает данные детей.

## Изменения политики

Политика может обновляться при изменении функций приложения. Актуальная версия должна публиковаться вместе с приложением или ссылкой в магазине.

## Контакты

Перед публикацией добавь контакт разработчика: email поддержки или страницу проекта.
