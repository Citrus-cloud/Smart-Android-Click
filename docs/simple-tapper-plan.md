# Simple Tapper Plan

## Шаг 1 — Минималистичный главный экран ✅

- Главный экран: метки, таймер, повторы, Запустить/Стоп.
- Убраны лишние технические экраны из основного пользовательского пути.

## Шаг 2 — Реальная плавающая метка ✅

- Добавлен overlay поверх других приложений.
- Реальные тапы идут через Accessibility Service.

## Шаг 3 — Premium minimal redesign ✅

- Новый чистый стиль: black/white, мягкие карточки, крупный hero-блок.
- Главный экран стал визуально проще и понятнее.
- Метки стали крупнее, контрастнее, с номерами.

## Шаг 4 — Несколько меток ✅

- В приложении можно добавить до 5 меток.
- В overlay можно добавить до 5 плавающих меток.
- Каждая метка двигается отдельно.
- При запуске приложение тапает по меткам по очереди.

## Шаг 5 — Расширенные функции: клик по картинке / иконке ⏭️

Следующий большой шаг:
- выбрать фото/иконку как шаблон;
- выбрать область поиска;
- выбрать точку тапа внутри найденной картинки;
- цикл: скрин → поиск похожей картинки → тап.
