# Simple Tapper Plan

Цель: убрать ощущение демо-конструктора и сделать приложение простым для реального теста.

## Шаг 1 — Минималистичный главный экран ✅

- Главный экран: только метка, таймер, повторы, Запустить/Стоп.
- Убраны сценарии, профили, аудит, safety review и лишние переходы из основного потока.
- Оставлены две вспомогательные кнопки: «Метка поверх» и «Расширенные».

## Шаг 2 — Реальная плавающая метка ✅

- Добавлен `FloatingTapperOverlayService`.
- Метку можно двигать поверх других приложений.
- Внутри overlay есть только Start/Stop и закрытие.
- Во время тапа overlay прячется на долю секунды, чтобы тап попадал в приложение под ним.

## Шаг 3 — Системные разрешения без внутренних согласий ✅

- Внутренние чеклисты и 10 подтверждений убраны из основного режима.
- Нужны только системные разрешения Android:
  - Accessibility Service — для настоящего тапа.
  - Overlay — для метки поверх экрана.

## Шаг 4 — Расширенные функции: клик по картинке / иконке ⏭️

Нужно добавить:
- выбор фото/скриншота как шаблона;
- выбор области поиска на экране;
- выбор точки тапа внутри найденной картинки;
- цикл: снять экран → найти похожую картинку → тапнуть в заданную точку.

## Шаг 5 — Полировка APK ⏭️

- Сохранение таймера / повторов.
- Режим бесконечного клика.
- Мини-кнопка аварийной остановки в overlay.
- Более компактный дизайн.
