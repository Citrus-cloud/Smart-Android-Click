# ClickFlow 0.2.0-beta — release notes

## Главное

Эта версия превращает ClickFlow из демо-прототипа в полноценную тестовую версию умного автокликера.

## Новое

### Простая тапалка

- Несколько меток.
- Таймер.
- Повторы.
- Бесконечный режим.
- Задержка старта.
- Сохранение настроек.
- Аварийная остановка.

### Overlay-метки

- Плавающие метки поверх других приложений.
- До 5 меток.
- Перетаскивание каждой метки.
- Сохранение позиций.
- Запуск/стоп из overlay-панели.

### Клик по картинке

- Добавление фото/иконки из галереи.
- Порог похожести.
- Точка тапа внутри шаблона.
- Область поиска.
- Один тап или постоянный режим.
- Поиск картинки на экране через MediaProjection.
- Реальный тап через Accessibility.

### Клик по тексту

- OCR через ML Kit.
- Поиск строки на экране.
- Contains/exact режим.
- Ignore case.
- Один тап или постоянный режим.

### Сценарий

- Простой сценарий по сохранённым overlay-меткам.
- Использует таймер, повторы и бесконечный режим из главного экрана.

## Что важно проверить

4. Overlay permission.
5. MediaProjection permission.
6. Обычные метки.
7. Overlay-метки.
8. Клик по картинке.
9. Клик по тексту.
10. Сценарий.

## Ограничения beta

- Image matching пока простой, без OpenCV и масштабирования.
- OCR зависит от качества текста на экране.
- Сценарии пока простые: только сохранённые overlay-метки.
