# ClickFlow — черновик описания для магазина

## Короткое описание

Умный автокликер для Android: тапы по меткам, картинке, тексту и пользовательским сценариям.

## Полное описание

ClickFlow — локальный автокликер для Android без root. Он помогает автоматизировать повторяющиеся нажатия: по фиксированным точкам, по найденной на экране картинке, по распознанному тексту или по пользовательскому сценарию из нескольких шагов.

### Возможности

- Overlay-метки поверх других приложений.
- Автоматические тапы по выбранным точкам.
- Клик по картинке или иконке, когда она появляется на экране.
- Клик по тексту через OCR.
- Сценарии из шагов: метка, фото, текст, пауза.
- Повторы, интервалы и бесконечный режим.
- Плавающая кнопка остановки.
- Локальное хранение настроек и сценариев.

### Приватность

ClickFlow работает локально на устройстве. Скриншоты, OCR, шаблоны изображений и сценарии не отправляются на сервер разработчика. Android Backup для локальных данных приложения отключён.

### Разрешения

ClickFlow использует Accessibility только для пользовательской автоматизации нажатий и сценариев, которые пользователь сам настраивает и запускает. Overlay нужен для плавающих меток и кнопки остановки. Захват экрана используется для локального поиска картинки или текста.
