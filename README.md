# ClickFlow Android

ClickFlow — минималистичный умный автокликер для Android.

## Возможности

- Обычная тапалка по одной или нескольким меткам.
- Плавающие overlay-метки поверх других приложений.
- Сохранение настроек и позиций меток.
- Таймер, повторы, бесконечный режим.
- Аварийная остановка.
- Клик по картинке/иконке.
- Клик по тексту через OCR.

## ✦ Премиум-режим «Сценарии»

Мощный режим автоматизации: пользователь сам собирает цепочку шагов и запускает её в один тап.

- **Типы шагов:** метка (тап по координатам), фото (поиск по шаблону), текст (OCR), пауза.
- **Цепочки шагов** с перемещением, дублированием и удалением.
- **Повтор всего сценария:** фиксированное число циклов или ∞.
- **Повтор и интервал на каждый шаг** отдельно.
- **Политика «если не найдено»** для каждого шага: пропустить, подождать и повторить, или остановить.
- **Своя библиотека сценариев:** создавайте, дублируйте и переключайте активный сценарий.
- **Плавающая панель прогресса** во время работы: текущий цикл, шаг и кнопка «■ Стоп».

Доступ к режиму открывается через экран-замок. Подписка появится позже — сейчас «Разблокировать Премиум» открывает всё бесплатно для тестирования.

## Разрешения

ClickFlow использует системные разрешения Android:

- Accessibility — для реальных тапов.
- Overlay — для меток поверх экрана.
- MediaProjection — для поиска картинки и текста на экране.

Внутренние лишние согласия и демо-чеклисты убраны из основного пользовательского потока.

## Архитектура сценариев

- `scenario/ScenarioModel.kt` — чистая Kotlin-модель (`Scenario`, `ScenarioStep`, `StepType`, `NotFoundPolicy`), покрыта юнит-тестами.
- `scenario/ScenarioLibraryStore.kt` — хранение библиотеки сценариев (JSON в SharedPreferences).
- `scenario/ScenarioActivity.kt` — премиум-билдер и экран-замок (Compose).
- `scenario/ScenarioEngineService.kt` — исполнение сценария + плавающая панель прогресса.
- `scenario/ScenarioPremium.kt` — флаг разблокировки и хук под будущую подписку.

## Как собрать

```bash
./gradlew testDebugUnitTest --no-daemon
./gradlew assembleDebug --no-daemon
```

```text

```
